# Space Defender
<img width="794" height="592" alt="image" src="https://github.com/user-attachments/assets/dd9ae583-f50c-477c-8270-525ebee8aa0d" />


Простая TDS на C++ с использованием SFML.

## Требования

- **CMake**: 3.28+
- **C++**: 17+
- **Компилятор**: MSVC 2022, GCC 9+, Clang 10+
- **SFML**: 3.0.2 (автоматически загружается)


## Управление

- **1, 2, 3, 4** - выбор башни
- **ЛКМ** - установить башню
- **R** - перезапуск
